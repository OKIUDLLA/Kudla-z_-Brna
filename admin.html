Zadejte GitHub Personal Access Token pro přímé nahrání změn na web. Token vytvoříte na github.com/settings/tokens s oprávněním repo.

Návod

Jak to funguje?

1. Upravte obsah v záložkách výše (koncerty, videa, fotky, shop, bio)

2. Změny se ukládají do prohlížeče (zůstanou i po zavření)

3. Pro publikování na web: buď stáhněte JSON soubory a nahrajte na GitHub, nebo použijte přímou aktualizaci s tokenem

4. Po nahrání na GitHub se web automaticky aktualizuje (obvykle do 1-2 minut)

Jak přidat fotky?

Fotky je potřeba nahrát přímo na GitHub do složky kudlazbrna-web/img/gallery/. Pojmenujte je jednoduše bez diakritiky a mezer.
