GitHub nastavení

Zadejte GitHub Personal Access Token. Token se zapamatuje v prohlížeči, nemusíte ho zadávat pokaždé. Token vytvoříte na github.com/settings/tokens s oprávněním repo.

Návod

Jak to funguje?

1. Upravte obsah v záložkách výše (koncerty, videa, fotky, shop, bio)

2. Změny se ukládají do prohlížeče (zůstanou i po zavření)

3. Klikněte na Publikovat na web (modrý pruh nahoře) nebo použijte záložku Export

4. Po nahrání na GitHub se web automaticky aktualizuje (obvykle do 1-2 minut)

Změna přihlašovacích údajů

Po změně údajů je potřeba Publikovat na web, aby se nové údaje projevily. Heslo je uloženo bezpečně jako SHA-256 hash — nikde se neukládá v čitelné podobě.
